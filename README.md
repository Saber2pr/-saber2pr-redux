# saber-redux

> Simplify of Redux in Typescript.

```bash
# from npm
npm install saber-redux

# from github
git clone https://github.com/Saber2pr/saber-redux.git
```

---

## start

```bash
# install the typescript and webpack
npm install
```

```bash
# auto compile to commonjs
npm start

npm run test
```
